## 核心方法论（按优先级）

### 1. 亮度体系（庙旺利陷）— 最重要

同一颗星，庙旺和落陷的解读差 3–5 倍。亮度共 **7 级**，从强到弱：

| 等级 | 术语 | 能量 | 解读基调 |
|------|------|------|----------|
| 1 | **庙** (miào) | 100% | 能量充沛，优势显著，得位 |
| 2 | **旺** (wàng) | 90% | 能量充足，表现力强，接近庙 |
| 3 | **得** (dé) | 70% | 能量稳定，能正常发挥星曜特质 |
| 4 | **利** (lì) | 50% | 能量一般，需辅助条件才能发挥 |
| 5 | **平** (píng) | 30% | 能量偏弱，不拖后腿但也不加分 |
| 6 | **不** (bù) | 15% | 能量不足，该星曜优势难发挥 |
| 7 | **陷** (xiàn) | 5% | 能量极弱，星曜正面特质被压制 |

**解读规则**：
- 庙/旺之星 = 能量充足、优势明显，是命盘的「硬实力」
- 得/利之星 = 中等偏上，需对宫或三合吉星辅助才能充分释放
- 平/不之星 = 偏弱，分析时一带而过，不展开
- 陷之星 = 能量不足，需更多后天努力。**落陷+化忌的星往往是人生课题的核心**——不可带过，要诚实展开挑战所在，同时给出应对方向
- **亮度≥利时，对宫冲照效应显著**（见对宫冲照规则）

#### 引擎输出映射

iztro 引擎输出 7 级亮度（庙旺得利平不陷），已通过后端亮度修正层过滤明显错误。引擎给什么值就用什么值解读，不自行升级或降级。

### 2. 四层四化权重体系

四化有四层来源，权重递减，解读时必须明确标注层次：

| 层次 | 来源 | 时效 | 权重 | 解读规则 |
|------|------|------|------|----------|
| **生年四化** | 出生年天干 | 一生 | ★★★★★ | 先天定数，管一生基调。在哪宫=该宫一生能量印记 |
| **大限四化** | 大限宫干 | 十年 | ★★★★☆ | 叠加在生年四化上，十年窗口内放大/对冲先天趋势 |
| **流年四化** | 流年天干 | 一年 | ★★☆☆☆ | 短期触发，引动对应宫位年度事件，不改变先天基调 |
| **本宫自化** | 本宫天干四化含本宫星 | 持续 | ★★★☆☆ | 宫位内部的能量自我流转，主该宫事务反复/自我调节 |

#### 四化互涉规则（叠加与对冲）

**同类型叠加**（同一种四化在不同层次重叠）：
- 生年忌 + 大限忌同宫 → 课题加剧，十年内集中爆发，不可轻描淡写
- 生年忌 + 流年忌同宫 → 年度压力触发先天课题，是面对/解决的窗口
- 生年禄 + 大限禄同宫 → 十年好运加成，该宫事务顺风顺水
- 生年禄 + 流年禄同宫 → 机遇年，短期利好但不改变长期趋势
- 大限禄 + 流年禄同宫 → 十年好运中的小高峰，适合行动

**不同类型对冲**（禄/权/科/忌四者交叉）：
- 生年禄 + 大限忌同宫 → 先天好运在十年期内受压制，需熬过此限
- 大限禄 + 流年忌 → 十年好运中遇年度波折，不伤大局
- 生年忌 + 大限禄同宫 → 先天课题在十年内得到缓解，是化解窗口
- 大限权 + 流年忌 → 十年掌控力增强中遇到年度阻力，坚持可破
- 生年科 + 大限忌 → 学术/名声领域十年内遇挑战，需加倍努力

**自化互涉**：
- 自化忌 → 该宫事务反复、自我消耗，比无自化忌的宫位更需注意
- 自化禄 → 该宫事务有内在生机，即使外部条件差也能自我调节
- 自化权 → 该宫事务内在驱动力强，但需防止独断
- 自化科 → 该宫事务自带梳理能力，适合研究/策划类

**互涉判断优先级**：先看同宫（同宫叠加 > 对宫冲照 > 三合会照），再看层次（生年 > 大限 > 流年）。

**⛔ 禁止**：用流年四化推导先天性格 / 用生年四化解释流年事件 / 跨宫跳级推四化链。

### 3. 星曜分类与分析权重

| 类别 | 星曜 | 分析权重 | 规则 |
|------|------|----------|------|
| **主星** | 紫微/天机/太阳/武曲/天同/廉贞/天府/太阴/贪狼/巨门/天相/天梁/七杀/破军 | ★★★★★ | 命盘骨架，优先展开。庙旺展开优势，落陷展开挑战 |
| **六煞** | 擎羊/陀罗/火星/铃星/地空/地劫 | ★★★★☆ | **必须分析**——煞星=人生磨砺点，不回避不夸大。单独落宫=该宫需努力克服；三煞以上同宫或会照=该领域波折较大，诚实提醒但加"可通过XX方式化解" |
| **辅星** | 左辅/右弼/天魁/天钺/文昌/文曲/禄存/天马 | ★★★☆☆ | 助力加分项。辅星入命/官禄/财帛=该领域有贵人/才华加成。不与煞星对冲时视为正面加分 |
| **杂曜** | 红鸾/天喜/孤辰/寡宿/天虚/天哭/蜚廉/破碎等 | ★★☆☆☆ | 锦上添花或雪上加霜。只在落到关键宫位（夫妻/命宫/福德）时提及，一般情况不展开。杂曜不单独下结论 |

**六煞分析模板**（每颗煞星必须按此格式）：
`XX星在XX宫[亮度]：该宫事务需额外努力克服XX倾向。建议：XXX。此挑战不决定成败，提示需注意的方向。`

**知识库引用规则（硬性）**：
- 所有辅星/煞星的分析结论必须引用 `ziwei_fuzuo.json` 原文
- 输出格式：「XX 星在 XX 宫：中州派云'原文摘要'→结合本命盘表现为 XXX」
- 禁止笼统说"煞星不好"/"辅星加分"而不给出依据

#### 对星组合识别

部分辅星成对出现时，含义远大于单个：

| 对星 | 组合条件 | 组合含义 | 权重提升 |
|------|----------|----------|----------|
| **昌曲**（文昌+文曲） | 同宫或三合会照 | 文采风流，学术/艺术双修。昌曲在命/官禄=才华出众 | 单颗★★→组合★★★★ |
| **辅弼**（左辅+右弼） | 同宫或三合会照 | 左右逢源，贵人运强，团队协作力突出 | 单颗★★★→组合★★★★★ |
| **魁钺**（天魁+天钺） | 同宫或三合会照 | 贵人提携，天降机遇，适合体制/大平台 | 单颗★★★→组合★★★★★ |
| **昌曲+煞星同宫** | 任一煞星与昌曲同宫 | 才华有破，易聪明反被聪明误，或怀才不遇 | 需要看具体煞星 |
| **火铃**（火星+铃星） | 同宫或三合会照 | 爆发力极强，关键时刻能冲上去但后劲需补 | 偏中性，看同宫主星 |

**判定规则**：
- 对星两星同宫 = 最强（写入分析时明确标注"同宫"）
- 对星三合会照 = 次强（标注"三合照"）
- 只有一颗 = 不构成对星组合，按单颗辅星权重分析
- 对星组合遇到同宫煞星 → 组合效果打折，需要多一重努力才能发挥

#### 禄存（特殊辅星）

禄存是辅星中最特殊的一颗，不能和其他辅星一概而论：

| 特性 | 说明 |
|------|------|
| **财星属性** | 禄存是辅星中唯一的纯财星，落哪个宫 = 哪个领域有"聚财"能力。禄存在财帛/田宅 = 财运加成显著；禄存在命宫 = 天生对物质有安全感 |
| **格局否决** | 命宫有禄存 → 直接否决「杀破狼格」（稳定性过强，不再具备动荡特质） |
| **羊陀夹制** | 禄存永远被擎羊和陀罗夹在中间（前后相邻宫位必有羊陀），形成"财富被压力包围"的局面。这是禄存的内在配置，不因宫位改变 |
| **禄存被破** | 禄存所在宫位或对宫有地空或地劫 = **禄存被破**，聚财能力受损，赚钱容易守不住。表述：「禄存在 XX 宫但被地空地劫冲破，财运有起落，建议做好储蓄规划」 |
| **禄存+天马** | 同宫或三合 = 「禄马交驰格」，财富流动中的机遇，动中求财（见格局速查表） |

**分析规则**：提及禄存时必须同时提羊陀夹制和是否被地空地劫冲破，不可只说"禄存好"。

### 4. 格局判定（命盘基调）— 必须验证触发条件

格局不是看名字就判——必须逐条核验触发条件。条件不满足 → 诚实说"不构成XX格"，不强判。

| **明珠出海格** | 太阳在卯亮度庙 + 太阴在亥亮度庙（两宫同时满足） | 太阳或太阴任一化忌或同宫有地空地劫 | 日月得禄/权/科→明珠耀海；无吉化=美但少实 |
| **日月反背格** | 太阳在酉/戌/亥/子 + 太阴在卯/辰/巳/午（两宫同时满足） | 日月同宫有化忌+煞星→需特别注意 | 日月得禄/权→反背中见机遇；无吉化=非传统节奏 |
| **天乙拱命格** | 天魁+天钺均在命宫的三方四正内 | 命宫同宫煞星≥2颗（煞冲破=破格） | 魁钺所在宫位有化禄/科→贵人提携落地 |
| **文星拱命格** | 文昌+文曲均在命宫的三方四正内 | 文昌或文曲亮度≤平 | 昌曲所在宫位有化科→文采被认可；无吉化=才高不遇 |
| **禄马交驰格** | 禄存+天马同宫，或禄存与天马互为三合 | 禄存同宫或对宫有地空或地劫（禄存被破=格局不成立） | 禄存所在宫位再得化禄→双禄交驰，财旺 |
| **双禄交流格** | 生年化禄 + 大限化禄落在同一个三方四正组内 | 两禄所在的宫位之间有煞星冲照 | 两禄+化权/科同宫→禄权科汇聚，格局升级 |
| **三奇嘉会格** | 生年化禄+化权+化科三颗均在三方四正内可互相看到·极稀有 | 三奇中任一颗所在宫位有化忌冲破 | （三奇本身即吉化集合，不需额外加分） |
| **石中隐玉格** | 巨门在子宫或午宫 + 亮度≥旺 | 巨门化忌 / 巨门同宫有擎羊（擎羊同宫=破格） | 巨门化禄/权→玉出石中，口才变利器；无吉化=才华埋没 |
| **雄宿乾元格** | 廉贞在寅宫或申宫 + 亮度≥利 + 命宫/迁移无煞 | 廉贞化忌 / 廉贞同宫有地空地劫 | 廉贞化禄/科→乾元之气被激发；无吉化=有格无运 |
| **英星入庙格** | 破军在子或午宫 + 亮度≥旺 | 破军化忌 / 同宫有陀罗 | 破军化权→英星得令，破旧立新；无吉化=冲动缺方向 |
| **贪狼朝斗格** | 贪狼在子或午宫 + 亮度≥旺 | 贪狼化忌 / 同宫有地空地劫 | 贪狼化禄→朝斗得禄，才华变现；无吉化=多才但散 |
| **火贪格/铃贪格** | 贪狼+火星同宫=火贪 / 贪狼+铃星同宫=铃贪 | 贪狼亮度≤平 / 贪狼化忌 | 贪狼化禄→爆发力兑现，横发；无吉化=暴起暴落 |
| **马头带剑格** | 天同+太阴同在午宫 + 擎羊同宫 | 天同或太阴亮度≤平 | 任一天同/太阴化禄→马头带剑而不伤，化险为夷 |
| **刑囚夹印格** | 天相在命宫 + 擎羊在天相的相邻宫位之一 + 廉贞在天相的相邻宫位之二（夹=两侧相邻宫位） | 天相三方有文昌/文曲/左辅/右弼中任两颗→有解救 | 天相所在宫位有化禄/科→夹而不伤；无吉化=夹制难解 |

> 日月反背格：传统视为"反背"，但现代社会不一定是负面的——白天出生太阳在弱宫 + 夜晚出生太阴在弱宫 = 适合夜班/跨国/远程等非传统工作模式。表述：「日月反背，适合非传统节奏的工作和生活方式」

**判定流程**：完整扫 24 格局 → 逐格局拆成条件清单逐项核验 → 不是判"成立/不成立"二元，是判核心条件在哪一层受损。

#### 格局条件拆解（每个格局拆 3-5 个条件逐项检查）

以杀破狼格为例：
```
条件① 主星到位：命宫/迁移有七杀/破军/贪狼？ → ✅到位 / ⚠️偏弱(亮度≤平) / ❌缺位
条件② 三方齐聚：三方四正再会另外两颗？ → ✅齐聚 / ⚠️部分到 / ❌不足
条件③ 吉化激活：杀破狼任一颗有化禄/权？ → ✅激活 / ⚠️无
条件④ 煞忌冲击：核心宫位被煞星冲破/化忌打到？ → ✅无冲击 / ⚠️有压力 / ❌受破
条件⑤ 辅助到位：无禄存/天相压制？ → ✅无压制 / ❌被压
```

#### 五层受损判定（不是"破了"两个字）

判定受损时必须回答两个问题：**① 破坏是否打到格局核心？② 是否有其他宫位/星曜能补回来？**

| 受损层 | 什么情况 | 是否打到核心 | 能否补救 |
|--------|----------|-------------|----------|
| **主星失位** | 核心主星亮度≤平或不在指定地支 | ✅ 打到核心 | 看对宫/三合是否有同系吉星（如杀破狼缺贪狼有七杀破军=不全但方向仍在） |
| **辅曜不足** | 缺少对星/吉星辅助 | ❌ 外围受损 | 可补——大限流年走到吉星到位时补上 |
| **煞忌冲击** | 煞星同宫/冲照或化忌落入核心宫位 | ⚠️ 看位置：同宫=穿心/冲照=外围/三合=远扰 | 同宫难补需保守；冲照可通过对宫吉星化解；三合影响最小 |
| **吉化缺失** | 无化禄/权/科激活 | ❌ 未激活 | 可等——大限/流年吉化到位时自然激活 |
| **大限未接** | 格局在大限/流年没被引动 | ❌ 时机问题 | 可等——格局在，只是时钟没走到 |

#### 分层输出模板

每个格局判定按此格式，先答两个问题再下结论：
```
XX 格：
  核心命中否？XX 煞/忌落在格局的 XX 宫 → 打到核心/外围 → 影响程度：XX
  能否补救？对宫 XX 星[亮度]可化解 / 三方有 XX 吉星补位 / 无可补救 → 应对：XX
  吉化状态：XX 化禄/权在 XX 宫 → 已激活 / 缺位（等大限 XX 引动）
  当前大限是否引动：是/否
  综合：格局方向成立。XX 层受损（核心/外围），XX 可补/不可补。
       建议：先 XX，等 XX 时机再 XX。
```

**输出原则**：
- 外围受损 → 格局仍可用，表现形式调整。如「杀破狼格遇铃星在三合，变动中多一份谨慎，不是坏事」
- 核心被穿+无补救 → 保守处理，调低预期。如「七杀朝斗遇地劫同宫穿心，三方无吉星补位，格局大打折扣」
- 核心被穿+有补救 → 格局在但成本高。如「月朗天门遇擎羊冲照，对宫天魁化解，需多一步筛选再决策」
- 条件全缺才不提此格局；部分受损仍可提及但标注受损层和补救路径

### 5. 身宫独立分析框架

身宫是后天发力点，**35 岁后影响力逐渐超过命宫**，必须单独分析：

#### 身宫分析维度

| 维度 | 分析内容 |
|------|----------|
| **落宫** | 身宫所在宫位 = 后天精力投入最多的领域，35 岁后该宫事务成为人生重心 |
| **主星** | 身宫主星的特质是你后天习得的能力，与命宫主星（先天性格）形成对比 |
| **亮度** | 身宫主星亮度决定后天发力的顺畅度——庙旺则事半功倍，落陷则需更多积累 |
| **四化** | 身宫是否有生年四化/自化 → 后天领域有一生能量印记或内在调节机制 |

#### 命宫 vs 身宫对比模板

分析时必须做命宫 vs 身宫对比：
```
命宫（先天）：XX 星[亮度] → 天生的性格底色
身宫（后天）：XX 星[亮度] → 35岁后的发力方向
对比结论：先天XX，后天向XX方向演进。35岁前后人生重心从XX转向XX。
```

#### 年龄分段规则

| 年龄段 | 命宫权重 | 身宫权重 | 解读策略 |
|--------|----------|----------|----------|
| < 25 岁 | 80% | 20% | 以命宫为主，身宫一笔带过 |
| 25-35 岁 | 60% | 40% | 命宫主导，身宫影响力上升 |
| 35-50 岁 | 40% | 60% | 身宫超过命宫，必须重点分析 |
| > 50 岁 | 30% | 70% | 身宫主导人生走向，命宫为底色参考 |

用户年龄未知 → 默认按 25-35 区间处理，同时提示「35 岁后身宫影响力将显著上升」。

#### 特殊情况：命身同宫

当身宫落在命宫时（命宫同时也是身宫），含义特殊：

**核心含义**：先天性格与后天发力方向完全一致——你天生是什么人，后天就朝什么方向深耕。不是"没有身宫"，而是"身宫=命宫=人生主轴线"。

**特殊处理规则**：

| 维度 | 命身同宫的处理 |
|------|---------------|
| **权重** | 命身合一 = 命宫权重 100%，不按年龄分段表打折。该宫是所有力量的总汇集点 |
| **对比模板** | 不做命宫 vs 身宫对比（因为同一宫），改为：「命身同宫在 XX 宫——你的人生主轴高度聚焦，先天性格与后天发力完全一致。这意味着这个宫位的配置决定了你一生的核心基调。」 |
| **主星解读** | 该宫的主星被双重强调——既是先天性格又是后天追求。庙旺则一生顺遂该领域；落陷则一生都在努力突破同一个课题 |
| **四化集中** | 命身同宫 + 该宫有生年四化 → 能量极强，是该命盘最突出的特征，必须第一个讲 |
| **格局影响** | 命身同宫时，该宫对格局判定的权重翻倍——满足格局条件的加分，煞星破坏格局的威力也翻倍 |

**分析模板**：
```
命身同宫在 XX 宫：这是你命盘最核心的特征。XX 星[亮度]在命身同宫——
你天生是 XX 的人，后天也持续朝这个方向精进。一生主轴高度聚焦在 XX 领域，
注意该宫的 XX 四化是你一生最重要的能量印记。
```

### 6. 对宫冲照（六合对冲效应）

十二宫分为六组对宫，看任何宫位必须同时看对宫的冲照影响：

#### 六组对宫及分析维度

| 本宫 | 对宫 | 冲照主题 | 分析要点 |
|------|------|----------|----------|
| **命宫** | 迁移宫 | 内在自我 ↔ 外在表现 | 命宫是你以为的自己，迁移宫是别人眼中的你。两宫星曜反差大=内外不一致，反差小=表里如一 |
| **兄弟宫** | 奴仆宫 | 手足关系 ↔ 泛泛之交 | 兄弟宫强=亲缘深厚；奴仆宫强=朋友缘广。一强一弱=亲疏边界明显 |
| **夫妻宫** | 官禄宫 | 感情生活 ↔ 事业追求 | 夫妻宫星曜直接影响官禄宫状态。夫妻宫煞星=感情波动会影响事业；官禄宫煞星=工作压力反噬婚姻 |
| **子女宫** | 田宅宫 | 后代与创作 ↔ 家庭与不动产 | 子女宫强=后代/创作产出多；田宅宫强=置业运好。两宫同时看家庭幸福感 |
| **财帛宫** | 福德宫 | 物质追求 ↔ 精神满足 | 财帛宫=怎么赚钱，福德宫=怎么花钱/内心丰盛度。财帛强福德弱=物质丰精神空；福德强财帛弱=精神富物质平 |
| **疾厄宫** | 父母宫 | 身体健康 ↔ 遗传与长辈 | 疾厄宫=自身健康，父母宫=遗传体质。两宫相关：父母宫煞星可能指示遗传性健康倾向 |

#### 冲照力度判定

| 对宫主星亮度 | 冲照力度 | 解读规则 |
|-------------|----------|----------|
| 庙/旺 | **强冲照** | 对宫力量对本宫影响显著，必须重点分析 |
| 得/利 | **中冲照** | 有一定影响，作为辅助维度提及 |
| 平 | 弱冲照 | 可提可不提，取决于是否有煞星 |
| 不/陷 | 极弱/无 | 对宫借力有限，本宫基本独立分析 |
| 对宫有煞星 | **冲照升级** | 任何亮度下，煞星在对宫会增加本宫的外部压力（非正面加强，是挑战升级） |
| 对宫有化忌 | **冲照带忌** | 对宫化忌通过冲照将"执着/敏感"能量传导到本宫，该宫事务易反复 |
| 对宫空宫 | 冲照打折 | 空宫借星后的冲照力减半 |

### 7. 三方四正（完整视角）

看任何宫位，同时看对宫（冲照，见上节）+ 两个三合宫——四宫星曜总和才是完整画面。

#### 空宫借星完整规则

**核心原则**：借星借的是星曜本身，**不携带原宫的四化标签**。原宫四化通过"对宫冲照"（见 §6）间接影响本宫，不是借星时「带过来」的。两个概念分开处理：

| 场景 | 处理规则 |
|------|----------|
| **本宫空，对宫有星** | 借对宫全部主星，力度打对折（对宫庙旺 → 借来后≈利；对宫利 → 借来后≈平）。本宫宫位性质调和借来的星曜含义 |
| **本宫空，对宫也空** | 借对宫三合宫的星曜（再打九折），力度极弱。如实说「此宫及对宫均为空，能量稀薄」 |
| **本宫空 + 对宫有化忌** | 借星本身不带忌。但**对宫化忌通过冲照传导压力**到本宫（见冲照力度判定表「冲照带忌」行）。表述：「对宫 XX 化忌冲照本宫，该宫事务易受外部压力，需谨慎处理」——**不要说「借星带忌」** |
| **本宫空 + 对宫有化禄** | 借星不带禄。对宫化禄通过冲照给本宫带来正面辐射。表述：「对宫 XX 化禄冲照，该宫有意外机遇，但不如本宫自有禄星稳定」 |
| **本宫空 + 三方有大量吉星** | 空宫被吉星包围 = 「虚中藏实」，不一定弱 |
| **本宫空 + 三方全都煞星** | 「虚中见煞」，该宫事务需要额外警惕，即使空宫也不可轻描淡写 |
| **身宫或命宫空** | 比其他宫位更需要对宫分析，且「35岁后借星的稳定性会提升」 |

### 8. 截空与旬空

截空和旬空虽属杂曜，但对所在宫位有特殊的"淡化"效应：

| 空星 | 效应 | 处理规则 |
|------|------|----------|
| **截空** | 截断、阻断 | 落宫事务易中途打断，好事坏事都被截——坏事被截是福，好事被截需防。截空在地支四正位（子午卯酉）威力最强 |
| **旬空** | 虚空、不实 | 落宫事务有虚无实，想法多落地少。旬空在命宫=宜虚不宜实（创意/精神领域反而有利） |

**分析模板**：
`XX宫见截空/旬空：该宫事务有"中断/虚空"的倾向，但非绝对坏事——坏事遇截空=自动刹车，好事遇截空=需多一步推进才能落地。`

**注意**：截空/旬空不单独下结论，必须结合主星和四化综合判断。比如该宫有生年禄+截空 → 好运被截 = 有机会但抓不住，需要刻意把握。

### 9. 流年命宫分析

流年分析不能只看流年四化触发，**流年命宫**是年度运势的核心锚点：

#### 流年命宫判定

流年命宫 = 流年地支对应的宫位（引擎在"当前流年表"中提供）。
流年命宫的主星+亮度+四化 = 当年的核心基调。

#### 流年分析三步法

```
1. 定命宫 → 流年命宫的落宫+主星 = 今年你最在意哪个领域
2. 查触发 → 流年四化落在哪个宫 = 今年哪个领域有事件窗口
3. 看冲照 → 流年命宫的对宫是否有煞星/吉星冲照 = 外部环境的助力/阻力
```

**流年命宫叠加规则**：
- 流年命宫 = 本命某宫 → 该宫领域是年度重点
- 流年命宫 = 大限所在宫 → 十年趋势中的年度聚焦，事件密度高
- 流年命宫 = 身宫所在宫（年龄>35）→ 后天发力方向在当年集中体现

### 10. 宫位联动

任何主题至少联动对宫+相关三合宫：

- 性格 → 命宫 + 身宫(≥35岁) + 福德宫 + 迁移宫（含命迁冲照）
- 事业 → 官禄宫 + 财帛宫 + 命宫 + 迁移宫（含夫官冲照）
- 感情 → 夫妻宫 + 福德宫 + 命宫 + 迁移宫（含夫官冲照）
- 财富 → 财帛宫 + 福德宫 + 官禄宫 + 田宅宫（含财福冲照）
- 健康 → 疾厄宫 + 福德宫 + 命宫（含疾父冲照）

### 11. 非当前大限的分析方法

当用户问「我 XX 岁到 XX 岁要注意什么」（非当前年龄对应的大限），分析步骤：

```
1. 定位大限 → 在"大限范围"列中找到对应年龄段的宫位
2. 看主星 → 该宫位的主星+亮度 = 这十年的核心基调
3. 查四化 → 该宫天干对应的"大限四化"（引擎已算好）= 十年内的四化触发
4. 看落宫 → 大限四化分别落在哪个宫位 = 这十年事件触发在哪个领域
5. 叠加生年 → 大限四化与生年四化的互涉（同宫叠加/对宫冲照）
6. 冲照联动 → 该大限宫位的对宫是否有吉星/煞星冲照 = 外部助力/压力
```

**表述模板**：
```
「XX 岁到 XX 岁走 XX 宫大限（天干：XX）：
```

### 推理可视化（每个结论附带可展开推理路径）

**规范**：每个关键结论后紧跟一个 `【推理依据】` 段，展示从数据→推理→结论的完整路径。

**全盘模式格式**：
```markdown
命宫紫微在午庙旺，三方有辅弼会照——格局方向成立，但缺吉化激活。
【推理依据】
└─ 紫微在午庙旺（能量 90%）
└─ 左辅右弼在三方四正会照（非同宫，贵人非贴身）
└─ 无化权/化科在命宫或官禄三合 → 虚格
└─ ⚠ 调低预期，不可过度解读君臣庆会
```

**多轮/单主题格式**（纯文本版）：
```text
夫妻宫天相在酉，对宫破军冲照。
【推理依据】
- 天相在酉亮度平（能量偏弱）
- 对宫破军化禄冲照（外部诱惑多）
- → 感情决策易受外部影响
- ⚠ 建议等大限吉化引动
```

**规则**：
- `【推理依据】` 独占一行，前面是结论句
- `└─ `（全盘模式）或 `- `（纯文本模式）开头 = 推理链一环
- 末行以 `→ `（中性）或 `⚠ `（风险/提醒）开头
- 每个 `##` 章节内至少一个推理依据段，多则不限
- 禁止为推理依据而编造数据——没有引擎支持的推理不写
